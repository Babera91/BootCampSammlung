CheatSheet

GIT

Git initialisieren

Erstellen des lokalen Git Ordners

Bsp

{ $ git init }

Zum Git hinzufügen

Datein zum Lokalem Gitordner hinzufügen

Bsp

{ $ git add . }

Naricht der Änderung

hinterlegt eine Message. Gleichzeitig puscht er den Ordner

Bsp.

{ $ git commit -m "Message" }

Hochladen in das Repository

schickt das lokale repository in das github repository

Bsp.

{ $ git push }

Anzeige der Commits

Zeigt alle Commit ausführlich an

Bsp.

{ $ git log }

Erstellung von Branches

Erstellt einen neuen Branch/Zweig

Bsp.

{ $ git branch “name”}

Aufzählung der Branches

Erstellt einen neuen Branch/Zweig

Bsp.

{ $ git branch -a}

Branchen Wechsel

Wechseln zwischen den Branches

Bsp.

{ $ git checkout}

Kopie eines Projektes

Kopiert ein fremdes Projekt von Github

Bsp.

{ $ git clone}

Zusammenführung von Branches

Führt 2 oder mehrere Branches zusammen

Bsp.

{ $ git merge}
