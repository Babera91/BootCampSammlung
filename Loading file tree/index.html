Terminal

CheatSheet

GIT

"Touch"

Man erstellt neue Ordner

Bsp.

{ $ touch index.html }

"mkdir"

Man erstellt neue Dateien

Bsp.

{ $ mkdir assets/html/css/js }

"cd"

Damit navigiert man im Verzeichnis

Bsp.

{ $ cd assets -> cd css cd css -> ../ cd assets }

"ls"

Zeigt den Inhalt im Verzeichnis

Bsp.

{ $ ls [Ordner: ´´X``] assets / index html }

"ls-a"

Selbe Funktion wie ls. Zusätzliche Einsicht in versteckte Dateien/Ordnern

Bsp.

{ $ ls -a] ./ Harz.png }

"cp"

Kopiert eine Datei/Ordner in ein anderes Zielordner

Bsp.

{ $ cp index.html assets/img }

"rm"

Kopiert eine Datei/Ordner in ein anderes Zielordner

Bsp.

{ $ cp index.html assets/img }

"rmdir"

Löschen von Ordnern

Bsp.

{ $ rmdir assets }
